# Weather App (Python)

This script fetches real-time weather data using OpenWeatherMap API.

## Features
- Get current temperature
- Weather condition
- Humidity

## How to Run
1. Add your API key in the code
2. Install dependencies:
   pip install requests
3. Run:
   python weather_app.py

## Sample Output
Enter city name: Kolkata

Weather in Kolkata:
Temperature: 30°C
Condition: clear sky
Humidity: 65%
